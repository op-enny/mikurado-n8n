# mikurado-n8n

n8n workflow definitions for **Mikurado — AI Dealer Operating System**.

Contains all automation workflows for agent orchestration, scheduling, and platform integrations.

Part of the [mikurado](../mikurado) umbrella repo.

## Quick start

1. Start n8n via Docker Compose from the umbrella repo
2. Open `http://localhost:5678`
3. Import workflows from the `workflows/` folder

See [CLAUDE.md](CLAUDE.md) for full guidance.
